# MSM AI Pack — Aide

Bienvenue dans l’aide de **MSM AI Pack** (DaDa:bit + WonderCam).

## Modèle de programme (pour n’importe quelle leçon)
- **Au démarrage** : `Lesson X → initialiser`
- **Toujours** : `Lesson X → lire … → décider → agir → pause 20 ms`

### Pourquoi ajouter `pause 20 ms` ?
- stabilise la lecture caméra (évite les mesures trop rapides)
- réduit les à-coups moteurs/servos
- améliore la fiabilité des détections (anti-rebond simple)
